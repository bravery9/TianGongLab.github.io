VQL 函数功能描述

功能函数名参数输出参数说明举例
找出 source 到 sink 之间的传输路径VQL.taintPropagationList<Long> source
List<Long>sink
Integer flag
List<Long>should
List<Long>shouldNot		通路上点的集合source 为 source 点 ID 集合，sink 为 sink 点 ID 集合，
should 为期望经过的点ID集合， shouldNot 为排除点ID的集合，
flag 默认为空为从source和sink中较少的一方传递 ，
flag=1 时为有方向从 source 到 sink；
flag=2 时为有方向从 sink 到 source；
flag=3 时污点分析为无向		CALL VQL.taintPropagation([845623], [845593, 845664], 1, [845658])
查找集合[845623]到集合[845593, 845664]之间的单向污点路径,并且途中需要经过集合[845658]
双向找出 source 到 sink 之间的传输路径VQL.doubleTaintPropagationLong sourceId
Long sinkId
Integer flag
List<Long> should
List<Long> shouldNot		通路上点的集合flag=1时为有方向；flag=2时为无方向;CALL VQL.doubleTaintPropagation(845623,845664,1)
同时从点845623和845664向中间查找有向污点路径。
使用最短路径找出 source 到 sink 之间通路VQL.simpleTaintPropagationList<Long>source
List<Long>sink
Integer flag		通路上点的集合flag=1时为有方向；flag=2时为无方向；flag=3时找出全部路径CALL VQL.simpleTaintPropagation([845623],[845664],1)
使用最短路径找出集合[845623]到集合[845664]的有向污点传输路径
获取当前项目所有基本块VQL.getAllBasicblocks基本块节点CALL VQL.getAllBasicblocks
找出当前分析项目中所有的基本块节点
获取当前项目所有函数VQL.getAllFunctions函数节点CALL VQL.getAllFunctions
找出当前分析项目中所有函数节点
获取当前项目所有文件VQL.getAllFiles文件节点CALL VQL.getAllFiles
找出当前分析项目中所有文件节点
获取指定code_line的AST路径VQL.getASTOfCodelineString function
Long nodeId
Long line		AST 路径function 为函数名，line 为行号，当 nodeId 为空时按照这两个属性进行查询，nodeId 为节点 ID，按照 code_line 的 id 进行查询CALL VQL.getASTOfCodeline("_mgfini_BFDD",11)
找出函数中第十一行code_line对应的ast路径
CALL VQL.getASTOfCodeline(31687)
找出code_line的id为31687对应的ast路径
获取指定函数的基本块VQL.getBasicblockOfFunctionLong functionId
String start_address		基本块节点start_address为函数入口点地址，表示函数名，二者选一输入即可CALL VQL.getBasicblockOfFunction(”0805be6b”)
找出入口地址为0805be6b函数的所有基本块
CALL VQL.getBasicblockOfFunction(22947)
找出id号为22947函数的所有基本块
通过变量被调用的函数和参数顺序定位变量VQL.getArgumentString callee
Long index		变量节点callee为调用函数，index为参数位置，
-1 代表输出，0 代表第一个参数、1 代表第二个参数、以此类推		CALL VQL.getArgument("malloc",0)
找出位于函数malloc第一个参数位置上的所有变量
获取指定节点指定层数的子节点路径VQL.getChildrenLong nodeId
Long stepStart
Long stepStop		到达子节点路径nodeId是指定节点的id，stepStart为最小跳数、stepStop为最大跳数CALL VQL.getChildren(958080,1,1)
找出距离节点958080一跳距离的子节点
通过入口地址查找函数节点VQL.getFunctionByAddressString entry_point函数节点fucntion 为指定函数的入口地址CALL VQL.getFunctionByAddress("080f316c")
找出入口地址为 0x080f316c 的函数
通过函数名称查找函数节点，支持模糊匹配VQL.getFunctionsByNameString function函数节点fucntion 为指定函数的名称CALL VQL.getFunctionsByName("zclient")
找出函数名为 zclient 的函数
获取指定文件的所有函数，支持模糊匹配VQL.getFunctionsOfFileString name函数节点name 为指定文件的名称CALL VQL.getFunctionsOfFile("bgpd")
找出文件名为 bgpd 下面的所有函数
通过变量获取其所在的语句节点VQL.getInfluencedCodelineLong nodeId
Long stepStart
Long stepStop		code_line 节点nodeId 是指定节点的 id，stepStart 为最小跳数、stepStop 为最大跳数CALL VQL.getInfluencedCodeline(150523, 1, 1)
找出距离变量节点 50523 步长为 1 的 code_line
获取节点所归属AST的输⼊变量VQL.getInputsOfCodelineLong nodeId变量节点nodeId 是指定代码行 code_line 的 idCALL VQL.getInputsOfCodeline(987013)
找出 code_line 987013 通过 asts 输入的相应变量
通过语句内容来获取code_line节点VQL.getCodelineByNameString namecode_line节点name是指定代码行code_line的内容CALL VQL.getCodelineByName("11: *DAT_080f1264 = iVar1;")
找出内容为 11: *DAT_080f1264 = iVar1 所对应的 code_line
通过基本块地址来获取 code_line 节点VQL.getCodelineOfBasicblockString start_address
String stop_address		code_line 节点start_address 是起始地址、stop_address 是终止地址CALL VQL.getCodelineOfBasicblock("0804e0f3", "0804e10d")
找出起始地址为: 0x0804e0f3，终止地址为: 0x0804e10d 的基本块对应的 code_line
通过函数名或者入口地址获取code_line节点VQL.getCodelineOfFunctionString function
String entry_point		code_line 节点function 是函数名、entry_point 是函数入口点CALL VQL.getCodelineOfFunction("_sub_I_01001_0.0", null)
找出函数名为 _sub_I_01001_0.0 对应的所有 code_line
CALL VQL.getCodelineOfFunction(null, "0804e645")
找出入口地址为 0x0804e645 的函数对应的所有 code_line
获取指定语句节点所归属AST的输出变量VQL.getOutputsOfCodelineLong nodeId变量节点nodeId为指定语句节点的id值CALL VQL.getOutputsOfCodeline(31683)
找出code_line31683通过asts输出的相应变量
获取指定节点指定层数的父节点路径VQL.getParentsLong nodeId
Long stepStart
Long stepStop		到达父节点路径nodeId是指定节点的id，stepStart为最小跳数、stepStop为最大跳数CALL VQL.getParents(31683, 1, 1)
找出距离节点31683步长为1的所有父节点
通过文件名获取文件节点VQL.getFileByNameString name文件节点name为指定文件文件名CALL VQL.getFileByName("bgpd")
找出文件名为bgpd的所有文件节点
通过命名空间获取类节点VQL.getClassByNameSpaceString namespace类节点namespace为指定的命名空间名称CALL VQL.getClassByNameSpace(”std::verify”)
找出命名空间名称为std::verify的所有类节点
获取指定类型的类节点VQL.getClassByTypeString type类节点type为制定的类类型CALL VQL.getClassByType(”User-defined”)
找出类类型为User-defined的所有类节点
获取指定名称的类节点VQL.getClassByNameString name类节点name为指定的类名称CALL VQL.getClassByName(”encryption”)
找出类名称为encryption的类节点
获取指定文件中的类节点VQL.getClassOfFileString name类节点name为指定的类名称CALL VQL.getClassOfFile(”bgpd”)
找出文件名为bgpd中的所有类节点
获取指定类中的成员函数VQL.getFunctionOfClassString name函数节点name为指定的类名称CALL VQL.getFunctionOfClass(”encr”)
找出类encryption中的所有函数节点
获取指定类中的成员变量VQL.getFileldOfClassString name变量节点name为指定的类名称CALL VQL.getFileldOfClass(”encryption”)
找出类encryption中所有的类成员变量
通过名称获取操作符VQL.getOperatorByNameString name操作符节点name为操作符的名称CALL VQL.getOperatorByName(”+”)
找出所有的+操作符变量
